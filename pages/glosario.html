Diccionario técnico

Conceptos clave para entender una reparación.

Estos términos son ideales para orientar al cliente antes de solicitar diagnóstico o autorizar una reparación.

OLED

Organic Light Emitting Diode

Tecnología donde cada píxel emite su propia luz. Permite negros profundos, gran contraste y excelente calidad de imagen.

QLED

Quantum Dot LED

Pantalla basada en puntos cuánticos para mejorar brillo, color y eficiencia sobre paneles LED tradicionales.

Backlight

Iluminación trasera

Sistema de luz que ilumina el panel LCD. Cuando falla, puede causar pantalla oscura, zonas apagadas o imagen apenas visible.

Main Board

Tarjeta principal

Es el cerebro del televisor. Controla entradas HDMI, sistema Smart, audio, imagen, memoria y procesamiento general.

T-Con

Timing Controller

Tarjeta que controla la sincronización de imagen del panel. Una falla puede provocar líneas, imagen dividida o colores incorrectos.

Fuente

Fuente de poder

Convierte la energía eléctrica para alimentar los circuitos internos. Una falla puede impedir el encendido del equipo.

Capacitor

Componente de almacenamiento

Ayuda a estabilizar energía dentro del circuito. Cuando se deteriora puede causar apagados, ruidos o fallas intermitentes.

HDMI

Entrada de audio y video

Conector digital usado para transmitir imagen y sonido. Puede dañarse por tirones, humedad o descargas eléctricas.

Firmware

Software interno

Programa base del equipo. Una actualización fallida puede causar reinicios, bloqueos o pérdida de funciones inteligentes.
